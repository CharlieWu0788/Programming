Project Title:
Catch Game

Description:
This project is a simple interactive game written in Jack. The player controls a paddle at the bottom of the screen using the left and right arrow keys. A box falls from the top of the screen, and the player must catch it. Each successful catch increases the score. Missing the box reduces the player's lives. The game ends when all lives are lost.

Controls:
- Left Arrow: move paddle left
- Right Arrow: move paddle right
- ESC: quit game
